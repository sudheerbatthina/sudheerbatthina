## Selected systems

### Sourcebound

An agentic knowledge system designed to produce useful, source-grounded answers over private documents. The system combines ingestion, hybrid retrieval, reranking, cited generation, caching, evaluation, and production-oriented APIs.

**Engineering evidence:** 163 passing automated tests, FastAPI service architecture, and a security-first product direction. Source code remains private while the product and repository history are prepared for a safe public case study.

[Product site](https://sourcebound.dev)

### RemaindALL

A privacy-conscious mobile product for location-aware reminders: save an errand or task and receive the reminder where it becomes actionable. Built across the mobile experience, permissions, geofencing, maps, and local notification flows.

**Current stage:** progressing toward App Store deployment. Source remains private during release preparation.

### StyleTrip.ai

A multimodal personal styling workflow that turns a photo plus context—occasion, weather, budget, fit, and preferences—into curated outfit directions and reusable style feedback.

**Engineering evidence:** typed Next.js application, passing typecheck and production build, Supabase-backed product architecture, and mock-safe development paths that avoid accidental paid-provider usage.
